# 🚗 Traffic Cascade — 蝴蝶效应模拟器

## 核心思想

> 一辆车的急刹，如何让三公里外的车堵成一锅粥？

交通阻塞的本质是**非线性波动的逆向传播**。当前车制动，后车必须响应——这个响应有延迟，延迟产生震荡，震荡放大为拥堵。一条蝴蝶翅膀的拍动（一次随意变道），足以让整个环线瘫痪。

## 物理模型：Intelligent Driver Model (IDM)

```cpp
a_desired = a_max * (1 - (v/v0)^4 - (s*/s)^2)
```

- `v` 当前速度
- `s` 与前车距离  
- `s* = s0 + v*T + v*Δv/(2√(ab))` 期望跟车距离
- 一旦 `s` 骤降（前方变道切入），`a` 从正值瞬间跌入负值
- 后续所有车辆依次经历这个过程，形成**逆向传播的冲击波**

## 蝴蝶效应放大机制

1. **初始扰动**：一辆车 ±5% 的随机速度波动
2. **非线性放大**：IDM 的四次方速度项让差异指数级增长
3. **空间耦合**：每一辆车的状态决定下一辆车的输入
4. **相变临界**：车流密度超过临界值后，小扰动永不消散

## 编译

```bash
g++ -std=c++17 -O2 -I/usr/include traffic.cpp -o traffic -lSDL2
./traffic
```

## 参数

- 车辆数：80
- 道路长度：1000m
- 最大密度：40辆/km（超过则进入同步流状态）
- 可调参数：加速度 `a_max`、安全距离 `s0`、车流密度

## 蝴蝶效应演示

按 **Space** 注入一次随机扰动（蝴蝶翅膀拍动），观察冲击波如何在车队中逆向传播并放大。
